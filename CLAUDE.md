# CLAUDE.md

> **Read [AGENTS.md](AGENTS.md) first — it is the canonical operating guide for this repo.**

This file serves as a pointer to the canonical project build guide. 

For development operations:
- **API (backend)**:
  - Run development server: `cd api && npm run dev`
  - Install dependencies: `cd api && npm install`
- **Platform (dashboard frontend)**:
  - Run development server: `cd platform && npm run dev`
  - Install dependencies: `cd platform && npm install`
- **Storefront (public frontend)**:
  - Run development server: `cd storefront && npm run dev`
  - Install dependencies: `cd storefront && npm install`
- **Test suite**:
  - Run API integration tests: `cd api && npm run test`
